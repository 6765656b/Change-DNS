Description
==============
This Script Change System DNS.

DNS Providers List:
- Cloudflare
- Google
- OpenDNS
- Quad9
- Shecan 
- Electrotm 
- Begzar 
- 403.online 
- radar.game

Usage
------
Add executable permission:
```shell
$ chmod +x Change_DNS.sh
```
Run the script:
```shell
$ ./Change_DNS.sh
```
